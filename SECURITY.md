# Security Policy

## Supported versions

Infrastack is currently in early development. Only the latest commit on the
`master` branch is supported with security updates.

| Version | Supported                      |
| ------- | ------------------------------ |
| 0.x     | :white_check_mark: latest only |

## Reporting a vulnerability

If you discover a security vulnerability in Infrastack, please report it
privately by emailing the maintainers. Do not open a public issue until the
vulnerability has been addressed.

Please include:

- A description of the vulnerability
- Steps to reproduce
- Affected versions or components
- Any suggested remediation

We aim to acknowledge reports within 72 hours and will work with you to verify
and resolve the issue responsibly.
